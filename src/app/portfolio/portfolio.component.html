Savetest - agile way to test your app

Savetest automates testing new features on your application or to test small command line applications. A problem with major test environment is the required knowledge of the platform, usually, this environment are controlled by QA and a developer has few tools immediately available for testing new features. I started this project in order to rapidly add new test cases as you program your application. The saved tests are stored as JSON objects making them easy to export if you want to reuse them.
